换行

尽量不要用\n\r这些有平台差异性的东西，可以使用 System.getProperty("line.separator") 来表示，返回的是一个String

Boolean

  • Boolean.getBoolean(String str)
    返回true当且仅当传入的参数是系统变量的key，并且value是”true”

  • Boolean.parseBoolean(String str)
    返回true当且仅当传入的参数非空，并且等于”true”，忽略大小写，返回boolean

  • Boolean.valueOf(String str)
    和parseBoolean逻辑一样，但返回的是Boolean的装箱类型

bufferreader释放内存

  1. new的时候有问题，出错误，第一个catch
  2. new完了运算有问题，出错误，第—个catch
  3. 运算有问题进入错误之后会导致new的内存无法释放，就要finally里close掉
  4. 如果本来就是null就没有分配内存，不能close
  5. close也可能出问题，第二个catch

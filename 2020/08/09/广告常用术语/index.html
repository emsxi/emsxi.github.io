广告常用术语

发表于 2020-08-09 分类于 广告

互联网广告行业变迁

  • 网络广告1.0

    • 门户时代（固定素材固定位置、根据CPM、CPD等计费）

  • 网络广告2.0

    • 搜索广告（根据查询文本相似性、根据CPC计费）
    • 网页搜索广告（根据网页内容文本相似性，根据CPC计费）

  • 网络广告3.0

    • 行为营销时代（根据用户的兴趣、各类计费模式）

广告计费模式

  • CPM (cost per mille)
    千次曝光计费
  • CPD (cost per day)
  • CPC (cost per click)
  • CPA (cost per action)
  • CPT (cost per time)
  • CPS (cost per sales)
    按销售额计费
  • CPT (cost per time)
    按时长计费
  • CPUV (cost per UV)

广告类型

  • 按目标

    • 品牌广告：以建立品牌知名度为目标的广告，一般关注的指标是广告曝光量，覆盖人群，TA%,互动情况等，对创意要求高，投放方式以合约、非标、招商为主，适合快消、汽车、数码等行业的客户
    • 效果广告：以促进直接转化为目标的广告，一般关注点击广告后直接带来的ROI，如下载、激活、付费、购买等，需要专业优化师维护账户，投放方式以竞价广告为主，适合游戏、电商、
      教育、网服等行业的客户

  • 按投放类型

    • 竞价广告：由用户自主投放，自主管理，通过调整价格来进行排名的广告，竞价广告每次面对广告请求都在线进行排序，胜出者获得曝光，不提前进行保量，一般竞价广告可选择的广告
      粒度比较粗
    • 合约广告：以合同的方式确定一次广告活动的投放总量和展示单价，售卖对象是广告位+人群，一般为品牌广告选择的投放方式

数据分析与统计

请求

  • PV (page view)
    网页被浏览的次数，PV 1:n query
  • UV (unique visitor)
    一天内一个页面独立访客数，UV 1:n PV
  • DAU (daily access user)
    类似UV，一般以用户id去重而不是设备或ip去重
  • query
    请求，query 1:(0/n) impression
  • QPS (query per second)
  • impression
    曝光，成功展示一个广告
  • click
    点击，点击一个广告
  • conversion
    转化，用户点击广告后进一步在广告主网站完成的有价值的行为，比如购买物品，下单等

链路

  • scoring
    主要完成广告质量计算和竟价过程，使用cpc、litectr、litecvr等对广告进行初步排序
  • ranking
    主要完成广告质量计算和竟价过程，使用cpc、pctr、pcvr、quality因子等对top(N)的广告进行reranking重排序

指标

  • CTR (click through rate)
    广告点击率，clicks / impressions * 100％
  • pCTR (predicted click-value rate)
    广告预估点击率，广告展现时发生点击行为的概率
  • CVR (conversion rate)
    转化率，从广告转化角度评估广告的展示效率，conversion（具体转化类型） / clicks
  • pCVR (predicted conversion-value rate)
    广告预估转化率，广告展现时发生转化行为的概率
  • eCPM
    • effective CPM
      实际计费的CPM价格（有效曝光价格），把各种计费模式的广告统一从曝光价格角度评估广告的展示效率，cost / impressions * 1000
    • expected CPM
      期望的CPM，一般指在一次广告排序中计算出的CPM价格，用于竞价广告排序
  • eCPC (effective CPC)
    实际计费的CPC价格（有效点击价格），cost / clicks
  • ROI (return on investment)
    投资回报率，(revenue - cost)/ cost
  • adload
    信息流广告中用于衡量用户看到广告的频率和密度，广告曝光数/（内容曝光数+广告曝光数）
  • quality
    广告质量因子，用来衡量广告质量的综合因子
  • 新鲜度
    从时间维度上保持用户对广告的吸引力，减少用户在短期内看到重复、相似的广告的概率，降低用户的审美疲劳

收益

  • RPM(revenue per thousand impressions)
    千次曝光利润，通常会被流量方用来评估不同的广告系统的赚钱效率，estimated earnings / impressions * 1000
  • GMV(gross merchandise volume)
    一定时间内网站成交金额，指拍下订单的金额，包含已付款和未付款部分，比实验成交金额大

出价与计费

  • bid
    广告出价，竞价后，实际收取的费用小于等于出价
  • GFP (generalized first price)
    一阶计费，竞争成功的广告主以竞价价格计费
  • GSP (generalized second price)
    二阶计费，第i位广告主以第i+1位广告主的出价加上0.01计费，这种出价方式使广告主更愿意以真实估价作为竞价价格，降低博弈成本，提升广告系统的收入和系统竞价稳定性

安装package control：

  1. 离线安装方法（离线配置文件）

    https://github.com/wbond/package_control 下载 zip 并重命名为 Package Control
    在菜单栏中点击 Preferences > Browse Packages…，打开一个文件夹，复制刚才的 Package Control 文件至此目录下
    在菜单中点击 Preferences，若出现 Package Settings 和 package control 选项，就说明安装 Package Control 成功了
    Preferences > Package Settings > Package Control > Settings - User 里面增加下面的值

安装 markdown 插件

快捷键ctrl + shift + p输入 install packages，安装 Markdown Preview 插件
打开 Preferences - Key Bindings，加上

令 markdown 渲染实时覆盖：

  1. 安装 LiveReload，安装之后ctrl + shift + p
  2. 输入 LiveReload: Enable/disableplugins
  3. 选择 Enable - Simple Reload

安装 json 格式化插件

快捷键ctrl + shift + p输入 install packages，安装 Pretty json 插件
默认格式化快捷键ctrl + alt + j

安装.proto高亮插件

快捷键ctrl + shift + p输入 install packages，安装 Protobuf Syntax Highlighting 插件
重启 sublime，并重新打开 .proto 文件生效

代码跳转插件

  1. 在 http://ctags.sourceforge.net 下载 ctags58.zip
  2. 本地解压，把 ctags.exe 放到 c:\windows\system32 目录
  3. 快捷键ctrl + shift + p输入 install packages，安装 ctags 插件
  4. 配置 ctags 路径，把 Package Settings 的 ctags default setting copy 到 user setting 中，并把 “commond” : “…/some/path” 修改为2中的路径
  5. 项目文件夹上右键点击 Ctags:Rebuild Tags 生成.tags文件索引（一段时间）
  6. 鼠标放在函数首字母左侧提示引用
